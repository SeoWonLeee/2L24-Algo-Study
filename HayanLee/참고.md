# Commit Convension

## 기존 컨벤션
- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서
- style: 포맷팅, 누락된 세미콜론 등
- refactor: 코드 리팩토링
- test: 테스트 관련
- chore: 기타 수정(작업중인 코드 포함)
- build: 빌드 시스템 또는 외부 의존성에 영향을 주는 변경
- remove: 파일을 삭제
- merge: 브랜치 병합

## Gitmoji 사용
- :sparkles: 새로운 기능 추가
- :bug: 버그 수정
- docs: 문서
- style: 포맷팅, 누락된 세미콜론 등
- :recycle: 코드 리팩토링
- test: 테스트 관련
- chore: 기타 수정
- :boom: 빌드 시스템 또는 외부 의존성에 영향을 주는 변경
- :fire: 파일을 삭제
- :twisted_rightwards_arrows: 브랜치 병합

<br>

- :construction: 코드 작업중
- :bulb: 주석 추가/수정
