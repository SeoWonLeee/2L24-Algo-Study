## **🧸  못생긴 수**

- 못생긴 수 : 오직 2, 3, 5만을 소인수 Prime Factor 로 가지는 수
<br/>

### **🚪 n번째 못생긴 수를 찾는 프로그램을 작성하세요.**

- 입력 조건 :
    - 첫째 줄에 n이 입력됩니다. (1 ≤ n ≤ 1,000)
- 출력 조건 : n번째 못생긴 수를 출력합니다.
- 입력 예시 : <br/>
    10
- 출력 예시 : 12
<br/>

### **🔑**

- 이전 못생긴 수들을 조합하여 새로운 못생긴 수를 생성한다.
- 이전에 생성된 못생긴 수들에 2, 3, 5를 곱한 값 중에서 가장 작은 값이 새로운 못생긴 수가 된다.
- DP 테이블을 이용하여 n번째 못생긴 수를 계산한다
